Device steps

How to install

iPhone

Safari home screen install

  1. Open the app link in Safari.
  2. Tap the Share button. If you do not see it, tap the three dots at the bottom right, then tap Share.
  3. Tap Add to Home Screen.
  4. Tap Add.

If you opened it inside Messages, Gmail, Instagram, or Chrome, copy the link into Safari first.

Android

Chrome install

  1. Open the app link in Chrome.
  2. Tap the menu button.
  3. Tap Install app or Add to Home screen.
  4. Open Student Command from the app icon.
Mac

Chrome or Edge install

  1. Open the app link in Chrome or Edge.
  2. Click the install icon in the address bar.
  3. Click Install.
  4. Launch it from Applications or Launchpad.
Windows

Desktop app install

  1. Open the app link in Chrome or Edge.
  2. Click the install icon in the address bar.
  3. Click Install.
  4. Pin it to Start or the taskbar.

What this is

PWA install

A PWA is a Progressive Web App: a website that can be installed and opened like an app. It is the right public version while Student Command is still proving whether students actually use it.

What this is not yet

Native installer

iPhone native distribution needs TestFlight or the App Store. Mac and Windows native downloads need separate signed builds. Those are later steps after real students test the free version.
